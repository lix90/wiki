corrr

corrr 一个用于探索相关关系的R包，提供了可视化的API。

install.packages("corrr")  # from CRAN
devtools::install_github("drsimonj/corrr")  # from github

forcats

forcats 一个用于操作分类变量／因子的R包。

install.packages("forcats")
devtools::install_github("tidyverse/forcats")

forcats::fct_recode() 对指定的因子的水平重新编码。

. %>%
  mutate(
    fct = fct_recode(fct, "level_label2" = "level1", "level_label2" = "level2")
  )
