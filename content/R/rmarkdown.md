# 头信息(front-matter)

头信息关键的参数为 `output`，用于指定文档格式。我个人常用文档格式：`html_document` `pdf_document` `revealjs_presentation`。

``` yaml
---
title:
author:
date:
output:
  html_document:
    toc: true
---
```
